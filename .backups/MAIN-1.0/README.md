# Дизайн MAIN 1.0 - Бэкап

Этот бэкап содержит оригинальный дизайн сайта до экспериментов.

## Дата создания
Создан: 24 декабря 2025

## Описание
Текущий стабильный дизайн с:
- Header с навигацией в центре (серый фон, черный слайдер)
- Sub-header с заголовком страницы и кнопками действий
- Высота header: py-4
- Высота sub-header: h-16

## Файлы в бэкапе
- `Header.tsx` - основной компонент шапки (src/components/Header.tsx)
- `styles.css` - глобальные стили (src/styles.css)
- `tasks.css` - стили страницы задач (src/tasks.css)

## Восстановление
Чтобы вернуться к этому дизайну, напишите: "давай вернем дизайн MAIN 1.0"
